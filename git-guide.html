Git Started (and GIT GOOD)

Brief Introduction on Git

Git is a Version Control System, commonly referred to as VCS. As the name suggests, it helps you keep track of versions of files and is often used by programmers to collaborate on software development.

'Git' is NOT 'Github'

Git is an open-source VCS tool used locally (offline). GitHub, on the other hand, is a cloud-based (online) VCS tool.

Why should you use GitHub?

GitHub allows you to keep track of versions of your code. If you make a mistake and need to undo it, you can retrieve a previous version if it's pushed to GitHub!

So how do I get started?

Windows

If you have Windows, download Git here.

Once installed, run git -v in Git Bash to check your installation.

Mac

Open your terminal and paste:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Then install Git using:

brew install git

Linux

If you're using Linux, Git is likely pre-installed. If not, use your package manager to install it.

Crucial Steps to Securely Connect Git with GitHub

  1. Create a GitHub account.
  2. Open your terminal and run:
ssh-keygen -t ed25519 -C "your_email@example.com"

If an error occurs, try:

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

Adding SSH Key to GitHub

  1. Run cat ~/.ssh/id_ed25519.pub and copy the key.
  2. Go to GitHub SSH Keys and add the key.

To test the connection, type:

ssh -T git@github.com

If successful, you'll see:

Hi Your_name_here! You've successfully authenticated, but GitHub does not provide shell access.
